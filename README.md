# 🛍️ shop.susubc.ca
 
Welcome to the **SUS UBC Online Shop**!  
This site is where students can browse and purchase official SUS merchandise like hoodies, crewnecks, jackets, patches, and more.  

---

## 🌟 Features (August 2025)
- Browse all current merchandise with images and descriptions
- Purchase items via Square links (Taken down as of September 2025 for maintenance)
- Responsive layout for both desktop and mobile  

---

## 🚀 Getting Started  

### Prerequisites  
- Node.js (>=18)  
- npm or yarn  

### Installation  
```bash
# Clone the repo
git clone https://github.com/ubc-sus/shop.susubc.ca.git

# Move into the project folder
cd shop.susubc.ca

# Install dependencies
npm install

# Run the development server
npm run dev
